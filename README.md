# Gaffgaff

Real-time chat application with AI interaction, featuring "Antigravity".

## Features

- **Real-time Messaging**: Instant communication using Socket.IO.
- **AI Integration**: Chat with "Antigravity", a simulated AI assistant that responds to your messages.
- **Responsive Design**: Modern, clean interface that works on various devices.
- **Dynamic UI**: Simulated typing delays and status updates for a realistic feel.

## Tech Stack

- **Backend**: Node.js, Express
- **Real-time Engine**: Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript

## Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory:
    ```bash
    cd gafgaaf
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

## Usage

1.  Start the server:
    ```bash
    npm start
    ```
2.  Open your browser and navigate to `http://localhost:3000`.
3.  Start chatting! Say "hello" or "who are you" to interact with Antigravity.

## Structure

- `server.js`: Main server file handling Express and Socket.IO logic.
- `public/`: Contains static frontend files (`index.html`, `style.css`, `script.js`).
